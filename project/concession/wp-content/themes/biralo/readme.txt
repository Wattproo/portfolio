== Description ==

A simple full-site editing blog theme. A full site editing theme is also known as a block theme. It is the fastest, easy to use, and customize theme. Perfect for writer, blogger, copywriter, poet, etc.

== FAQs ==
1. How can I edit/customize the theme?
Go to Appearance >> Editor and click on the section you want to change. 

2. Is Biralo Compatible with WooCommerce Plugin?
Yes, you can make eCommerce site using this theme and WooCommerce plugin.

3. Do you offer custom themes and plugins development?
Yes, we do offer that. Please contact us via contact page.
